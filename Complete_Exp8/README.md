# 🚀 EXP8 - Student Management REST API
This project is a Spring Boot REST API for managing student data with CRUD operations.

## 📌 Base URL
http://localhost:8080/api/students

## ⚙️ Features
- Add student
- Get all students
- Get student by ID
- Delete student
- JSON request/response

## 🛠 Tech Stack
Java, Spring Boot, Maven, REST API, Postman

## ▶️ Run
mvn spring-boot:run
